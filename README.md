# Robot Baseplate (Bodenplate)

This repository contains a custom-designed robot baseplate intended for small mobile or modular robots.

# Dimensions
- *Length:* 80 mm  
- *Width:* 100 mm  
- *Thickness (Depth):* 2 mm  

# Design Features
- Multiple mounting holes for motors, sensors, or brackets
- Slot openings for cable management or adjustable mounting
- Symmetrical layout for balanced assembly
- Lightweight flat design suitable for 3D printing or CNC cutting



## Manufacturing
- Recommended material: PLA / PETG / Aluminum (depending on use)
- Suggested print orientation: Flat on bed
- No supports required
